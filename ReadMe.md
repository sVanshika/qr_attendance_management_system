# QR Based Attendance Management System
To start frontend:

1. `cd qrms_frontend/attendance-app`

2. Add your device's IP in .env file  - 172.17.49.85
`npm start --host <172.17.49.85>;`
Application is live on http://172.17.49.85:3000/

To start backend server:

1. `cd qrms`
2. `./mvnw spring-boot:run`
